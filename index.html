nanoc by ddfreyne

nanoc is a site compiler written in Ruby for building awesome web sites.

This is the Git mirror of the nanoc Mercurial repository.

Dependencies

No dependencies necessary for basic usage, although you may want to additionally install some or all of these extras: coderay rubypants builder w3c_validators rack erubis haml RedCloth bluecloth markaby maruku rdiscount mime-types rainpress

Install

gem install nanoc
